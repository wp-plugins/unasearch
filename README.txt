Unasearch is a simple plugin for improving content search on Wordpress.

== Description ==

Unasearch is a new plugin in development that aims to improve the Wordpress search engine simply.
It is developed by Unacode, freshly created by a french fan of Wordpress. The purpose of Unacode is to share the skills acquired on Wordpress over the years, including through the development of plug-in.
Feel free to download and review, all feedback are welcome!

Here is the list of features in Unasearch:

* Plugin in two languages : English and French.
* Choose which post types to include in the search.
* Choose the orderby of the search query.
* Choose the order of the search query.
* Choose the post status of the search query.

== Installation ==

This section describes how to install the Unasearch plugin and get it working.

1. Upload 'unasearch' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "Unacode" and change settings to your needs.

== Frequently Asked Questions ==

= Unasearch is it compatible with other advanced search plugins ? =

Unfortunately not. Unasearch changes the Wordpress requests and other search plugin could conflict with this work.
